# MicroLeague Coin (MLC)

The utility token powering the next generation of sports simulations, creator leagues, and fan rewards — built on Base blockchain.

## Getting Started

### Prerequisites
- Node.js 18+ & npm

### Installation

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite (build tooling)
- Tailwind CSS (styling)
- Framer Motion (animations)
- Recharts (data visualization)
- shadcn/ui (component library)

## Project Structure

```
src/
├── components/     # Reusable UI components
│   ├── layout/     # Header, Footer
│   ├── sections/   # Landing page sections
│   ├── dashboard/  # User dashboard components
│   ├── creator/    # Creator dashboard components
│   ├── modals/     # Modal dialogs
│   ├── presale/    # Presale widget
│   └── ui/         # shadcn/ui primitives
├── hooks/          # Custom React hooks
├── pages/          # Route page components
├── lib/            # Utility functions
└── assets/         # Static assets
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run preview` — Preview production build

## Deployment

Build and deploy the `dist/` folder to any static hosting provider (Vercel, Netlify, Cloudflare Pages, etc.)
